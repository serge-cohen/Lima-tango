Properties Description

LimaDetector Class

Revision: - Author:



Device Properties
Property name
Property type
Description
DetectorDescription Tango::DEV_STRING Detector user-defined text to identify the engine.
DetectorType Tango::DEV_STRING Define the type of the connected Detector .
Availables types :
- AdscCCD
- BaslerCCD
- MarCCD
- Pco
- PerkinElmer
- PilatusPixelDetector
- ProsilicaCCD
- PrincetonCCD
- SimulatorCCD
- XpadPixelDetector
DetectorPixelDepth Tango::DEV_USHORT Define the pixel depth of the detector :
Availables values :
- 8
- 16
- 32
DetectorVideoMode Tango::DEV_STRING Define the format of video stream:
Availables values :
- NONE
- Y8
- Y16
- Y32
- Y64
- RGB555
- RGB565
- RGB24
- RGB32
- BGR24
- BGR32
- BAYER_RG8
- BAYER_RG16
- I420
- YUV411
- YUV422
- YUV444

Nota:
This property is usefull only for detectors having video capabilities.
Otherwise, use only NONE.
FileFormat Tango::DEV_STRING Define the format of image files :
Availables values :
- EDF
- NXS
- RAW
- TIFF
FilePrefix Tango::DEV_STRING Define the prefix used for the image files name.
FileIndexPattern Tango::DEV_STRING Define the pattern of the index used for image files names.

Nota :
%04d mean that file name index will be end with exactly 4 digits filled by '0' if necessary .
FileNbFrames Tango::DEV_LONG Define the amount of frames stored in the target file.
If Nexus file, this is the NbAcqPerFile.
FileTargetPath Tango::DEV_STRING Define the Path where Files will be generated, only when savingFile is checked.
DebugModules Array of string Define modules that we need to have some debug traces.
Availables values :
- None
- Hardware
- Control
- Common
- Camera
DebugLevels Array of string Define Lima verbose level of traces.
Availables values :
- Fatal
- Error
- Warning
- Trace
- Funct
- Param
- Return
- Always
DebugFormats Array of string Define Lima traces format.
Availables values :
- DateTime
- Thread
- Module
- Obj
- Funct
- FileLine
MemorizedRoi Array of short Memorize/Define the Region of Interest of the Acquisition:
origin X
origin Y
Width
Height

Nota:
if any roi value is <0, then we consider all detector area as Roi.
MemorizedBinningH Tango::DEV_USHORT Memorize/Define the Binning Horizontal value of the Acquisition
MemorizedBinningV Tango::DEV_USHORT Memorize/Define the Binning Vertical value of the Acquisition
MemorizedAcquisitionMode Tango::DEV_STRING Memorize/Define the acquisitionMode attribute at Init device
Availables values :
- SINGLE
- ACCUMULATION
MemorizedTriggerMode string Memorize/Define the triggerMode attribute at Init device
Availables values :
- INTERNAL_SINGLE
- EXTERNAL_SINGLE
- EXTERNAL_MULTI
- EXTERNAL_GATE
- INTERNAL_MULTI
- EXTERNAL_START_STOP
- EXTERNAL_READOUT
MemorizedShutterMode Tango::DEV_STRING Memorize/Define the shutterMode attribute at Init device
Availables values :
- MANUAL
- AUTO_FRAME
- AUTO_SEQUENCE
MemorizedShutterOpenTime Tango::DEV_DOUBLE Memorize/Define the shutterOpenTime attribute at Init device
MemorizedShutterCloseTime Tango::DEV_DOUBLE Memorize/Define the shutterCloseTime attribute at Init device
MemorizedExposureTime Tango::DEV_DOUBLE Memorize/Define the exposureTime attribute at Init device
MemorizedExposureAccTime Tango::DEV_DOUBLE Memorize/Define the exposureAccTime attribute at Init device
MemorizedLatencyTime Tango::DEV_DOUBLE Memorize/Define the latencyTime attribute at Init device
MemorizedNbFrames Tango::DEV_LONG Memorize/Define the nbFrames attribute at Init device
MemorizedFileGeneration Tango::DEV_BOOLEAN Memorize/Define the fileGeneration attribute at Init device
MemorizedFlipX Tango::DEV_BOOLEAN Memorize/Define the flipX attribute at Init device
MemorizedFlipY Tango::DEV_BOOLEAN Memorize/Define the flipY attribute at Init device
